
# 向僵尸开炮小游戏自动化挂机脚本

本项目是一套基于 Python 开发的游戏自动化工具，利用图像识别技术（OpenCV）与屏幕模拟操作（PyAutoGUI）实现全自动挂机。主要包含“环球救援抢位”和“精英主线自动推图”两大功能。

## 📋 功能概览

- **环球救援助手 (`qiang_huan.py`)**：实时监控聊天频道，自动点击招募信息并抢占救援名额。
- **精英主线助手 (`zhuxian_main.py`)**：循环挑战精英关卡，支持自动开局、自动释放技能、超时自动退出及体力耗尽自动终止。
- **通用工具库 (`utils.py`)**：
    - 核心图像识别引擎（单目标与多目标匹配）。
    - 自动处理网络重连、非法授权弹窗、战斗结束返回。
    - 智能技能选择逻辑（根据图片库优先级自动选技能）。
    - 兼容 PyInstaller 打包后的路径识别。

## 🛠️ 环境准备

1. **Python 环境**：建议使用 Python 3.8+
2. **安装依赖**：
   ```bash
   pip install pyautogui opencv-python numpy pillow
      ```
  
### 🖥️ 窗口设置
* **窗口尺寸**：脚本适配的游戏窗口尺寸为 **542 * 1008**。可以根据自己当前的游戏窗口尺寸，截相应的图片，替换picture和zhuxian_picture文件夹中的图片。
* **显示要求**：运行脚本前，请确保模拟器或游戏窗口可见，且不要被其他非透明窗口遮挡。
* **注意**：每次运行保持窗口大小不变，避免对图片文件夹中的图片更换，带来工作量。并且可能存在版本更新后，需要对picture和zhuxian_picture文件夹中的图片进行更换。

### ⚙️ 核心配置说明
你可以在各脚本文件的头部修改以下全局变量：

| 变量名 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `limit_hours` | 19 | 脚本运行的最长时间（小时），超过后自动停止。 |
| `region` | `(0, 0, 600, 1040)` | 屏幕识别区域 `(x, y, 宽, 高)`，请根据你的模拟器实际坐标进行调整。 |
| `limit_cishu` | 10^9 | 限制最大运行次数，达到后退出。 |
| `maximum_time_one_round` | 600 | (主线脚本) 单局最长允许时间（秒），超时则强制退出防止挂机卡死。 |

### 📂 文件结构
* **`qiang_huan.py`**: 抢救援脚本。启动前请先进入游戏内的聊天页面或救援页面。
* **`zhuxian_main.py`**: 精英主线脚本。启动前请先进入精英模式关卡选择页。
* **`utils.py`**: 存放所有底层逻辑、图像识别算法和公共 UI 处理函数（如自动重连等）。
* **`picture/`**: 通用图片库目录，存放通用按钮、技能图标、异常状态截图。
* **`zhuxian_picture/`**: 主线专用图片库目录，存放暂停、退出、主线特定状态截图。

### 🚀 使用指南

#### 1. 抢环球救援
* **操作步骤**：
    1. 打开游戏，切换到 **任意关卡的精英模式页面** 或 **任意级别环球救援页面**。
    2. 运行脚本：
       ```bash
       python qiang_huan.py
       ```
* **逻辑说明**：脚本会循环执行：`检查是否入队` -> `点击聊天` -> `点击招募` -> `抢票`。

#### 2. 精英主线挂机
* **操作步骤**：
    1. 打开游戏，切换到 **精英模式关卡页面**。
    2. 运行脚本：
       ```bash
       python zhuxian_main.py
       ```
* **逻辑说明**：脚本会自动执行：`点击开始` -> `识别开局` -> `自动选技能` -> `结算返回`。
* **超时保护**：如果单局超过 10 分钟未结束，脚本会自动点击暂停并强制退出，重新开始下一轮。

#### 3. 通用自动处理 (utils)
无论运行哪个脚本，系统都会自动监控并实时处理以下情况：
* **技能选择**：识别到预设技能图标或“已激活”按钮时，将自动进行连点操作。
* **异常弹窗**：自动识别并处理“拒绝授权”、“断网重连”、“网络干扰”等弹窗，确保挂机不中断。
* **体力检查**：识别到“体力不足”状态时，脚本将自动停止运行并安全退出。

### ⚠️ 注意事项
* **路径问题**：所有图片资源必须严格存放在脚本同级目录下的 `picture/` 或 `zhuxian_picture/` 文件夹内。
* **防卡死机制**：在 `qiang_huan.py` 中，每尝试抢票 100 次，脚本会**强制休息 10 秒**。此时你可以趁机移动鼠标夺回控制权或关闭程序。
* **匹配精度**：如果出现识别不灵敏的情况，可以微调代码中各图片定义元组末尾的 `confidence`（置信度）数值（范围 0 到 1）。
* **安全免责**：本脚本仅供技术交流与学习使用，请勿用于违反游戏服务协议的场景。使用脚本产生的任何后果及风险由使用者自行承担。
